# Banco de Dados B*

Uma implementação de uma base de dados simples utilizando como algoritmo de indexação uma [Árvore-B*](./ARVORE.md). Feito como um trabalho para a disciplina de Organização de Arquivos.

## Organização das pastas
- ```src```: arquivos .c
- ```header```: arquivos .h
